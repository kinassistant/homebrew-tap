# KIN Homebrew Tap

Homebrew tap for KIN command-line tools.

```sh
brew install kinassistant/tap/kin
```

The cask in `Casks/` is generated and pushed automatically by GoReleaser when a
`v*` release is tagged in [kinassistant/kincalendar](https://github.com/kinassistant/kincalendar)
(see `frontend/cli/.goreleaser.yaml`). Do not edit by hand.
